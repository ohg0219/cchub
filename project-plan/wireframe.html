📋 Section 01

프로젝트 개요

Claude Code Starter Kit을 발견하고, 미리보고, 한 번에 설치하는 마켓플레이스

서비스 이름 후보

cckit.dev starterkit.sh claudekit.dev agentkit.dev kitshare.dev claude-starters.dev

핵심 컨셉

기존 Skills 마켓플레이스(skills.sh, skillsmp.com 등)는 개별 SKILL.md 파일을 공유합니다. 하지만 실제 개발자가 필요한 건 "내 프로젝트에 맞는 전체 세팅"입니다.

우리 사이트는 Skills + Hooks + Agents + CLAUDE.md를 하나의 "스타터 킷"으로 묶어 원클릭 설치하는 경험을 제공합니다. npm의 create-next-app처럼 npx cckit install spring-boot-enterprise 한 줄로 프로젝트 전체 AI 인프라가 세팅됩니다.

차별화 포인트

핵심 차별화
패키지 단위 공유
개별 스킬이 아닌 Skills + Hooks + Agents + CLAUDE.md를 하나의 "킷"으로 묶어 공유. 프로젝트 타입별 완성형 세팅을 한 번에 설치.
핵심 차별화
Hooks 최초 전문 지원
기존 사이트 중 Hooks를 전문적으로 다루는 곳이 없음. 빌드 검증, 린트, 보안 스캔 등 자동화 Hooks를 공유하고 설치하는 최초의 플랫폼.
시장 기회
한국 개발자 타겟
한국어 UI + Java/Spring Boot 생태계 특화. 기존 사이트는 전부 영어권 + React/Node.js 위주. 네이버/카카오 API, MyBatis 등 한국 실무 기술 지원.
UX 차별화
인터랙티브 프리뷰
설치 전 킷의 파일 구조, 각 Hook/Skill의 동작 방식, 설정 의존관계를 시각적으로 미리 확인 가능. "어떤 파일이 어디에 설치되는지" 투명하게 공개.

핵심 기회 영역

Gap 1
Hooks 공유 플랫폼 전무
Claude Code Hooks (PreToolUse, PostToolUse, Notification 등)를 공유하는 전용 플랫폼이 전혀 없음. 빌드 자동 검증, 린트 실행, 금지 파일 수정 방지 등 실무에서 매우 유용한데 발견이 어려움.
Gap 2
조합 최적화 부재
skillhub.club에 "Stack"이 있지만 Skills만 묶은 것. Skills + Hooks + Agents + CLAUDE.md의 유기적 조합, 즉 "이 Skill이 이 Hook과 어떻게 연동되는지"를 보여주는 곳이 없음.
Gap 3
비영어권 / Java 생태계 무시
기존 사이트 모두 영어 + React/Node.js/Python 중심. 한국 기업 현실인 Spring Boot + MyBatis + Gradle + 한국어 API는 완전히 소외됨.

🛠 Section 05

기술 스택 추천

빠른 론칭과 한국 개발자 타겟에 최적화된 조합

웹사이트 (프론트엔드 + 백엔드)

★ 추천

Next.js + Vercel

빠른 MVP 출시에 최적. SSR/SSG 자유자재, Vercel 무료 배포, 글로벌 CDN.

  • Next.js 15 (App Router)
  • TypeScript
  • Tailwind CSS
  • Vercel 배포 (무료 티어로 시작)
  • 한국어/영어 i18n: next-intl

Astro + Cloudflare

정적 콘텐츠 위주라면 더 가볍고 빠름. 마크다운 기반 콘텐츠에 강점.

  • Astro 5
  • Cloudflare Pages (무료)
  • 빌드 시간이 매우 빠름
  • 동적 기능 추가 시 제약

Spring Boot (직접 구축)

Hayden의 주력 스택. 하지만 웹 프론트엔드 생산성에서 Next.js 대비 느림.

  • Spring Boot 3 + Thymeleaf
  • 배포 인프라 직접 구성 필요
  • MVP 속도에서 불리
  • 장기적으로 API 서버로 전환 가능

데이터베이스

★ 추천

Supabase (PostgreSQL)

무료 티어 충분. Auth 내장, Realtime 지원, REST API 자동 생성.

  • 킷 메타데이터, 사용자, 설치 통계
  • GitHub OAuth 로그인
  • Row Level Security

PlanetScale (MySQL)

MySQL 선호 시. 무료 티어, 브랜칭 DB 기능이 유용.

  • MySQL 호환 (Spring과 궁합)
  • 스키마 브랜칭으로 안전한 마이그레이션

CLI 도구 (npx cckit)

★ 추천

Node.js CLI (npm 패키지)

npx cckit install ...로 즉시 실행. npm 생태계 활용, skills.sh와 동일한 UX.

  • Commander.js (CLI 프레임워크)
  • Inquirer.js (인터랙티브 프롬프트)
  • chalk (터미널 스타일링)
  • npm 레지스트리에 배포

추천 조합 요약

웹사이트: Next.js 15 + Tailwind + Vercel DB: Supabase (PostgreSQL) + GitHub OAuth CLI: Node.js + Commander.js → npm 배포 킷 저장: GitHub 리포지토리 (킷 = Git repo) 검색: Algolia 또는 Supabase Full-text Search 분석: Vercel Analytics + Plausible (프라이버시) i18n: next-intl (한국어 우선, 영어 확장)
💡 이 조합의 장점: 무료/저비용으로 시작 가능, MVP 2-3주 출시 가능, 글로벌 확장 용이
🗓 Section 06

개발 로드맵

MVP부터 글로벌 확장까지

Phase 1 — MVP 런칭
2~3주
랜딩 페이지 + 킷 목록/상세 페이지 + CLI 도구 기본 기능 (install 명령어). 직접 만든 spring-boot-enterprise 킷을 시드 콘텐츠로 등록. 한국어 UI + GitHub OAuth 로그인.
Phase 2 — 커뮤니티 기반
1~2개월
킷 등록 기능 (GitHub repo 연결 → 자동 분석). 별점/리뷰 시스템. 설치 수 트래킹. 한국 개발자 커뮤니티(velog, disquiet, okky)에서 초기 사용자 확보. Hooks 전용 탐색 필터 + Hook 동작 시각화.
Phase 3 — 영어 확장
2~3개월
영어 UI 추가. React/Next.js, Python, Rust 등 다양한 생태계 킷 확보. 킷 제작자 프로필 + 팔로우 기능. AI 기반 킷 추천 ("내 프로젝트 package.json 분석해서 추천").
Phase 4 — 플랫폼 확장
3~6개월
팀/조직용 프라이빗 킷 공유 (유료 기능). 킷 버전 관리 + 자동 업데이트 알림. MCP 서버 통합 지원. Claude Code 외 다른 에이전트 (Cursor, Copilot, Windsurf) 정식 지원.

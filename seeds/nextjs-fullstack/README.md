# nextjs-fullstack

> Next.js 15 App Router + TypeScript + Tailwind CSS 풀스택 개발 스타터 킷

## 소개

Next.js 15 App Router 기반 풀스택 프로젝트를 위한 Claude Code 스타터 킷입니다.
Server Component vs Client Component 선택 기준, 데이터 페칭 패턴, Tailwind CSS 규칙을 포함합니다.

## 포함 구성

| 타입 | 이름 | 설명 |
|------|------|------|
| Skill | nextjs-patterns | Next.js 15 App Router 패턴 가이드 |
| CLAUDE.md | - | Next.js 프로젝트 개발 규칙 |

## 설치

```bash
npx cckit install nextjs-fullstack
```

## 요구사항

- Node.js >= 18
- Next.js >= 15
- Claude Code

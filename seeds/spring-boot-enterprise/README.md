# spring-boot-enterprise

> Java Spring Boot + MyBatis + Thymeleaf 엔터프라이즈 개발용 AI 스타터 킷

## 소개

Spring Boot 기반 엔터프라이즈 백엔드 개발을 위한 Claude Code 스타터 킷입니다.
레이어드 아키텍처, MyBatis 매퍼 작성, Java 코딩 컨벤션에 대한 AI 가이드를 제공합니다.

## 포함 구성

| 타입 | 이름 | 설명 |
|------|------|------|
| Skill | spring-boot | Spring Boot 아키텍처 및 패턴 가이드 |
| Skill | mybatis | MyBatis 매퍼 작성 가이드 |
| Skill | java-conventions | Java 코딩 컨벤션 |
| CLAUDE.md | - | Spring Boot 프로젝트 개발 규칙 |

## 설치

```bash
npx cckit install spring-boot-enterprise
```

## 요구사항

- Java >= 17
- Spring Boot >= 3.x
- Claude Code
